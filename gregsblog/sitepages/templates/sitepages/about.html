About This Blog





The inspiration for this blog project came from my desire to improve my skills in programming. The basics of it were taken from an online course taught by Nick Walter and I highly recommend his material.

This blog site was built using Django/Python, with assistance from HTML, CSS and Bootstrap. So, this was a full-stack project (of sorts) and it was certainly fun to get some exposure to both the front and back-end. Ubuntu, nginx and gunicorn were used to deploy the site.

Although this blog started out as a coding challenge and learning opportunity, I plan to keep the blog going with mainly programming related posts. The project was a challenge at times and I will go into more detail about those challenges in some of the blog posts. Also, I hope to add functionality to the Blog itself as I improve my skills.

Thanks for reading this far!!
